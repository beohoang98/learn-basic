# LEARN BASICS PROJECT

## Simple Webserver

Learn to make a simple webserver with C++11 on Linux Socket Library

Enviroment: Linux

## Build example

### Requirement

- CMake

```bash
cmake CMakeList.txt
make
```

## Run
```bash
cd ./web
./index
```

## Clean
```bash
make clean
```

Website is run on *http://localhost:8000*
